ShotLingua プライバシーポリシー

本プライバシーポリシー（以下「本ポリシー」といいます。）は、アプリ開発者（以下「当社」といいます。）が提供する日本語学習アプリ「ShotLingua」（以下「本アプリ」または「本サービス」といいます。）における、ユーザーに関する情報の取扱い方針を定めるものです。
本サービスを利用する前に、本ポリシーをよくお読みいただき、内容に同意のうえでご利用ください。

第1条（適用範囲）

1. 本ポリシーは、本アプリおよびこれに付随するサービスにおいて、当社が取得・利用するユーザー情報の取扱いに適用されます。

2. 本アプリからリンクされる第三者のウェブサイト、サービス、アプリケーション等における情報の取扱いについては、各事業者のプライバシーポリシーに従うものとし、本ポリシーは適用されません。

第2条（取得する情報）

当社が本サービスを通じて取得する情報の主な例は、以下のとおりです。

1. アカウント・設定情報（原則ローカル保存）

  • ローカルユーザーID（端末ごとに生成されるUUID）
  • ログイン状態フラグ
  • 表示言語設定、母語設定
  • オンボーディングアンケートの回答（日本語レベル、学習目的、学習時間帯等）
  • サブスクリプションプラン情報（無料／Pro／Super 等）

2. 学習データ・コンテンツ（原則ローカル保存）

  • 単語カード・単語帳（語彙、読み、意味、例文、学習ステータス、デッキ名等）
  • 学習進捗（各単語の復習回数、最終学習日、正答率等）
  • AIチャット履歴（AIキャラクターとの会話テキスト）
  • 日記・メモリー投稿（ユーザーが入力した文章、およびこれに基づき生成された語彙・例文）
  • スピーキング練習結果（発話テキスト、AIによるフィードバック、エラーパターン等）
  • 成長ログ・週次分析（学習時間、回数、達成したマイルストーン等）

3. 画像・音声・位置情報

  • カメラで撮影した写真（マジックカメラ機能用。アプリ内のキャッシュ領域に一時保存され、写真ライブラリには保存されません）
  • マイクから取得した音声（スピーキング練習、音声通話等）
  • 端末の位置情報（「使用中のみ」取得。単語やメモリーに場所タグとして紐付ける目的で使用）

4. プロファイル情報

  • ユーザーが任意で入力するプロフィール（よく出てくる人物名、場所、テーマ等。AIのパーソナライズに利用）

5. ローカルアナリティクス情報

  • 画面遷移・ボタン操作等に関するイベントログ（アプリ改善のため、端末内のJSONファイルとして保存。現時点ではサーバー送信なし）

6. デバイス情報（必要最小限）

  • OSバージョン、アプリバージョン等、本サービスの提供・不具合調査に必要な範囲の情報

第3条（情報の保存場所）

1. 端末内に保存される情報

  • 多くの学習データ（単語帳、学習進捗、チャット履歴、日記等）は、ユーザーの端末内（UserDefaults、ファイル、Keychain等）に保存されます。
  • OpenAI APIキーはKeychainに暗号化して保存され、当社が内容を閲覧することはできません。
  • ローカルアナリティクスログはJSONファイルとして端末内に保存され、現時点では外部送信されません。

2. サーバーに送信される情報

  • AI機能や音声認識等の提供のため、後記第4条に定める外部サービス提供者のサーバーに、一部の情報が送信される場合があります。
  • 現時点で、当社が独自に運用するバックエンドサーバー（ユーザーアカウントを集中管理するサーバー等）への送信は限定的または無効化されており、多くのデータは端末ローカルに留まります。

第4条（外部サービスへの送信と利用目的）

本サービスでは、以下の外部サービスを利用してAI機能等を提供するため、必要な範囲でユーザー情報が送信されることがあります。

1. OpenAI API

送信される情報の例：

  • テキスト：チャットメッセージ、日記・メモリーの内容、例文、スピーキング練習の書き起こし、ユーザープロファイル情報（よく出てくる人物名・場所・テーマ等）
  • 画像：マジックカメラで撮影した写真（画像内の物体認識・語彙生成のため）
  • 音声：音声通話やスピーキング練習で録音された音声（リアルタイム会話・発話解析のため）

利用目的：

  • 日本語学習に関する回答の生成
  • 単語帳・例文・クイズの自動生成
  • 発話内容の解析・フィードバック
  • 写真からの語彙抽出

データの取扱い：

  • OpenAIによるデータの保存期間、モデル学習への利用の有無等は、OpenAIのプライバシーポリシーおよび利用規約に従います。
  • ユーザーは、OpenAIのポリシーを確認の上で本機能を利用してください。

2. Appleの音声認識サービス

  • AppleのSpeechフレームワークを利用する場合、音声データがAppleのサーバーで処理されることがあります（デバイスや設定によりオンデバイス処理の場合もあります）。
  • 利用目的は、日本語音声の認識およびスピーキング練習機能の提供に限られます。
  • データの取扱いは、Appleのプライバシーポリシーに従います。

3. 位置情報サービス（Core Location）

  • 位置情報は、「どこでその単語を覚えたか」といった学習の文脈を記録する目的でのみ利用されます。
  • 位置情報は端末内に保存され、当社のサーバーや第三者へ送信されることは想定していません。

第5条（クッキー等の利用）

本アプリ自体は、Webブラウザにおけるクッキーと同等の技術を直接利用していませんが、外部サービス（OpenAI、Apple等）が独自にクッキーその他の識別子を利用する場合があります。これらの取扱いについては、各事業者のポリシーをご確認ください。

第6条（利用目的）

当社は、取得したユーザー情報を、主として以下の目的で利用します。

  1. 1. 本サービスの提供・維持・改善のため
    • 日本語学習機能（単語帳生成、フラッシュカード、クイズ等）の提供
    • AIチャット・音声通話・スピーキング練習等の提供
    • バグ修正、性能改善、新機能の開発
  2. 2. ユーザーに最適化された学習体験の提供のため
    • ユーザーのレベル・興味・学習履歴に応じたコンテンツの出し分け
    • 苦手な単語の分析および追加教材の提案
  3. 3. お問い合わせ対応のため
    • 不具合報告・問い合わせ内容の確認、返信のため
  4. 4. 不正利用の防止・対策のため
    • 規約違反行為の調査・対応
    • セキュリティインシデントの検知・防止

第7条（ユーザー情報の第三者提供）

  1. 1. 当社は、次のいずれかに該当する場合を除き、ユーザー情報を第三者に提供しません。
    • ユーザー本人の同意がある場合
    • 法令に基づく場合
    • 人の生命、身体または財産の保護のために必要であり、本人の同意を得ることが困難である場合
    • 公衆衛生の向上又は児童の健全な育成の推進のため特に必要がある場合であって、本人の同意を得ることが困難である場合
    • 国の機関等への協力が必要な場合
  2. 2. 前条で記載した外部サービスへの情報送信は、第三者提供ではなく、当社が本サービスの提供のために利用する「業務委託先への提供」に該当します。

第8条（安全管理措置）

  1. 1. 当社は、ユーザー情報への不正アクセス、紛失、破壊、改ざん、漏洩等を防止するため、必要かつ適切な安全管理措置を講じます。
  2. 2. 本アプリにおいては、KeychainによるAPIキーの暗号化保存、ファイル保護機能の利用（完全保護属性）等を通じて、ユーザー情報の安全管理に努めます。
  3. 3. ただし、インターネット通信の性質上、完全な安全性を保証するものではありません。ユーザーご自身においても、端末のロック設定等、適切な管理を行ってください。

第9条（ユーザーによる情報の管理・削除）

  1. 1. ユーザーは、本アプリ内の設定画面等から、学習データや一部ユーザー情報をリセット・削除できる場合があります。
  2. 2. 本アプリをアンインストールすると、端末内に保存されている多くのデータは削除されます。ただし、外部サービス（OpenAI等）のサーバーに保存されたデータまで自動的に削除されるものではありません。
  3. 3. ユーザーが自身の情報の開示・訂正・削除等を希望する場合は、アプリ内のお問い合わせ窓口よりご連絡ください。合理的な範囲で対応を検討します。

第10条（未成年者の利用）

  1. 1. 本サービスは、原則として13歳以上のユーザーを対象として設計されています。
  2. 2. 13歳未満の未成年者が本サービスを利用する場合には、必ず保護者の同意を得た上で利用してください。
  3. 3. 未成年者による利用に関する責任は、保護者が負うものとします。

第11条（本ポリシーの変更）

  1. 1. 当社は、必要に応じて、本ポリシーを変更することがあります。
  2. 2. 本ポリシーを変更する場合、当社は、変更後の内容および効力発生日を本アプリ内での表示その他の適切な方法で周知します。
  3. 3. 効力発生日以降にユーザーが本サービスを利用した場合、変更後の本ポリシーに同意したものとみなされます。

第12条（お問い合わせ窓口）

本ポリシーおよびユーザー情報の取扱いに関するお問い合わせは、本アプリ内のお問い合わせフォーム又は当社が別途指定するメールアドレスまでご連絡ください。
